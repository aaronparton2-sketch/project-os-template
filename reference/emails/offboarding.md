# Offboarding Email

When a client decides to leave. Be professional, make the handover clean, and leave the door open.

---

## Client Leaving

**Subject:** Wrapping up — everything you need for [Business Name]

**Body:**

Hi [Name],

No worries at all — I appreciate you letting me know, and I'm glad I could help get [Business Name] set up.

Here's everything you need to take over:

**Your website code**
- GitHub repo: [GitHub repo link]
- You have full access — this is yours, you own it completely
- It's built with [React/HTML/etc.] and hosted on [Netlify/Vercel]

**Your domain**
- [domain.com.au] is registered on GoDaddy
- I can transfer the domain to your own GoDaddy account, or point it wherever you need — just let me know
- Domain is paid up until [renewal date]

**Your hosting**
- Currently on [Netlify/Vercel] under my account
- To move it: you can create your own [Netlify/Vercel] account (free) and deploy the code from GitHub
- I'm happy to help with the migration if you need a hand

**Your CRM / automations** (if applicable)
- Google Sheet CRM: [Sheet link]
- n8n workflows will stop running once I deactivate them
- If you want to keep the automations running, I can help you set up your own n8n account

**Final billing**
- Your last payment was [date] — no further charges will be made
- I'll cancel the Stripe subscription today

If you ever need anything down the track, the door's always open. Happy to help anytime.

All the best with [Business Name],
Aaron
Mycelium AI

---

## Notes for Aaron

- **Don't burn bridges.** Today's leaver is tomorrow's referral (or return customer).
- **Transfer everything cleanly.** Domain, code, sheet access — don't hold anything hostage.
- **Cancel Stripe subscription immediately** after sending this
- **Update CRM:** Set lifecycle stage to Churned, add notes on why they left
- **Deactivate n8n workflows** for this client
- **Optional:** Ask for a Google review or testimonial before they go ("If you had a good experience, a quick Google review would mean a lot")
